就在昨天，Mac OS X已经和Classic Mac OS一样年纪了

作者：蓬岸 Dr.Quest
知乎文章编号：37250815
创建于：2018-05-24 1:33:26
修改于：2018-05-24 1:33:25

2018年5月23号是一个特殊的日子。这一天是Mac OS X 10.0在2001年3月24日发布之后的第17年一个月零29天——6269天，一个奇怪的数字——但它正好也是1984年1月24日（初代Mac发布的日期）到2001年3月24日所经过的时间。

也就是说，Mac操作系统的第二个时代，属于Mac OS X（也就是现在所说的macOS）的时代，所存续的时间已经和它之前的一个时代一样长了。

当然，我需要做一些解释说明：Mac OS X有过一个公开测试版，Mac OS 9则要到2002年才正式被送进坟墓。Classic模式则保留在Mac OS X当中，直到Mac OS X 10.5 Leopard中才被删除。

不过这仍然是一个里程碑。让我非常想知道Mac的下一步将会成为什么样子。我怀疑我们即将看到新的Mac OS带来一次地震般的变革——我们很可能会看到一系列的变化让我们理解中的Mac变为某种不同的东西，而这或许会受到iOS的成功所带来的影响。

可能时间是对的。苹果每个十多年就会更换一款芯片，而下一次的变革也可能正在来临。这一切在之前就已经发生过，而在之后仍会发生。
