作者：Dan Knight

1996年3月10号，随着初代PalmPilot的闪亮登场，Palm终于解决了从1992年就开始困扰他们的问题——为别人的硬件开发软件（参见本系列的上一篇：Palm简史（一）PalmPilot的诞生）。这一次，Palm有了他们自己的硬件和软件，并最终让Palm品牌成为PDA（个人数字助理）的代名词。

第一代产品：Palm OS 1

经过特殊设计的PalmPilot不仅适合手持使用，而且刚好能放进裤子口袋，这让用户能够更加方便的随身携带。

第一代的PalmPilot有两个型号，入门级的Pilot 1000有128KB系统内存，而高端的Pilot 5000则有512KB，两款机型都使用了摩托罗拉68328龙珠（DragonBall）CPU，这款16Mhz的低功耗处理器是32位68000的近亲，而68000也正是早期Mac电脑所使用的处理器。68328处理器基本上是一个单片机，理论上PalmPilot最多可以支持12MB的系统内存和4MB的ROM，但最初的设备只能支持到1MB。

这两款设备都使用了Palm OS 1.0，屏幕分辨率为160x160像素，并有一个单独的Graffiti触控笔输入区。

很快Palm就被百乐文具（Pilot Pen Company）起诉商标侵权，在法庭败诉之后，Palm将其商品名“Palm Pilot”改为“PalmPilot”（不带空格），并在之后简化为Palm。

早期的PalmOS设备只有一种方式与其他硬件设备通信，即历史悠久的RS232串口协议。

第二代产品：Palm OS 2

1997年3月10号，在初代产品发布一年之后，Palm发布了PalmPilot Personal（个人版）和PalmPilot Professional（专业版）。新设备使用了Palm OS 2.0，并且添加了屏幕背光。个人版安装有512KB内存，而专业版则有1MB。

可选的配件包括了底座、14.4Kbps调制解调器，和带有红外线通信功能的内存升级模块，红外线功能很快就在用户中间流行起来，因为它可以让两部PDA之间以无线连接传递信息。

PalmPilot的内存升级套件，可以将内存升级到2MB，并加入了红外通信功能。

第三代产品：Palm OS 3

时间又过了一年，1998年3月，内置红外线通信功能的Palm III进入人们的视野。Palm的第三代产品搭载了Palm OS 3.0操作系统和四级灰度触摸显示屏，而内存则升级到了2MB。

此外我们也会注意到Palm III的下边缘经过了重新设计，只有些许弧度，这和之前版本明显的圆弧状下边缘有着明显的不同。

Palm III的红外线通信的一个重要用途，就是用来配合带有红外线接口手机上网

Palm OS 3也是第一个授权给其他设备制造商的版本，最早的第三方制造商是1998年的Symbol和Handspring，不过直到1999年月Handspring才推出他们的第一款设备。在本系列的下一篇我们将详细讲述Handspring背后的故事。

Palm IIIx作为Palm III的增强版本于1999年发布，在Palm III的基础上将内存提升到4MB，并使用了速度更快的摩托罗拉CPU——龙珠EZ，这让Palm IIIx的速度比之前的设备都快。Palm IIIx带有一个内置扩展槽，可以进行升级或是添加新的功能。而Palm OS 3.1则包含了许多对Palm OS 3.0问题的修正，此外新的Flash ROM也可以让用户在未来对系统进行升级。

发布于1999年晚些时候的Palm IIIe则是Palm IIIx的廉价版本，它没有使用Flash ROM，因此系统无法升级。这是唯一一款没有使用Flash ROM的Palm设备，它和Palm IIIx一样使用了龙珠EZ处理器，但只有2MB内存。

发布于1999年2月的Palm V同样使用了改进后的龙珠EZ CPU，带有2MB内存和16级灰度触摸屏。Palm V使用了氧化铝外壳，并且是第一款内置充电电池的Palm设备，它同样预装了Palm OS 3.1系统。

Palm VII则是第一款内置无线通信天线的Palm设备，它使用了Mobitex无线通信服务，月租$14.95，Palm VII内置2MB内存，操作系统则是Palm OS 3.2。

Palm VII和Nokia E61

1999年10月推出的Palm Vx在内存容量上卖出了一大步，成为第一款带有8MB内存的Palm。它还是第一款主频高于16Mhz的Palm，使用了20Mhz的龙珠EZ处理器，它也是第一款使用Palm OS 3.3的设备。

2000年2月，Palm推出了它们的第一款彩色PDA，Palm IIIc，这款机型使用了8-bit（256色）触摸屏。

自给自足

在个人电脑领域存在着两种商业模式。一种模式以苹果和Commodore为代表，同时设计自己的硬件，并开发自己的操作系统。而另一种模式则是以8-bit时代的CP/M电脑和IBM兼容机为代表，硬件制造商购买其他公司的操作系统授权。

Palm Computing最初采用的是软硬通吃的策略，而从1998年的Palm OS 3开始，他们开始将操作系统授权给其他制造商。

Palm曾经是USRobotics的子公司，之后3Com收购USR，因此Palm也成为3Com的子公司。2000年3月1号，3Com旗下的Palm Inc.在纳斯达克独立上市。

第四代产品：Palm OS 4

2000年8月，Palm发布了他们的入门级产品线Palm m100系列，m100使用了2MB内存和16Mhz龙珠EZ，而配置更高的m105则在m100的基础上将内存扩大到8MB。

迈克尔·乔丹特别版Palm m100

之后的m125使用了33Mhz的龙珠VZ CPU，8MB内存，和SD卡扩展槽，操作系统也升级到Palm OS 4.0，接下来的m130搭载了Palm OS 4.1系统，并在m125的基础上升级为16位彩色屏幕。

2001年3月6号，就在Palm Inc.从3Com中独立之后的第一天，Palm m500系列被公之于众，这是第一批支持SD卡槽的Palm设备，采用的是Palm OS 4.0系统。

Palm m500使用了33Mhz龙珠VZ处理器，8MB内存，和4-bit，16级灰度160x160像素的黑白触摸屏。而m505则是Palm第一款16-bit彩屏PDA，支持超过6万5千种颜色，相比8-bit，256色的Palm IIIc迈进了一大步。

一年之后，2002年3月4号，Palm推出了m500的改进版本m515，新设备使用了Palm OS 4.1系统，内存增大了一倍达到16MB，屏幕背光也有所改善。

作为Palm最新的平价产品，白色设计的Palm Zire于2002年圣诞季发布。为了节约成本，Zire使用了没有背光的单色（无灰阶）显示屏，16Mhz龙珠EZ处理器，以及和大多数早期型号类似的2MB内存，这款机型采用了Palm OS 4.1系统，但没有配置SD卡槽。

Palm m505和Zire，可以看到Zire上的中文品牌名“奔迈”

授权Palm OS

2002年1月，Palm Inc.成立了全资子公司PalmSource，新公司的主要业务是开发和授权Palm OS，但实际上Palm从1998年就已经开始授权Palm OS了。

分庭抗礼

1998年6月，Palm Computing的一部分成员另起炉灶成立了Handspring，并成为第二家获得Palm OS授权的厂家，同时也可能是Palm OS授权厂家中最成功的一家。
