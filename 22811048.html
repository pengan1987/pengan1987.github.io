BLIPTASTIC!

《BLIPTASTIC!》是8 Bit Weapon成员电脑娘（ComputerHer）Michelle Sternberger的第三张个人专辑，收录了9首使用老式电脑硬件制作的8-bit曲目：

1) SIDuction - Commodore 64, Game Boy, 老式鼓机


2) Outer Space - Commodore 64, Game Boy, Atari 800 XL, 任天堂NES, 老式鼓机

3) Software - Commodore 64, Game Boy, 苹果II, Omnichord, 老式鼓机


4) Club Microwave - Commodore 64, Game Boy, Quasimidi Sirius, 老式鼓机


5) Bliptastic! - Commodore 64, Game Boy, 任天堂NES, 老式鼓机

6) Halloween Party - Commodore 64, Game Boy, 老式鼓机


7) Viking - Commodore 64, Game Boy, 老式鼓机


8) Futurism - Game Boy, 老式鼓机


9) Sugar Cube Remix* - Commodore 64, 老式鼓机

参与制作的乐器除了大家都比较熟悉的Commodore 64、Game Boy、任天堂NES这些常用的Chiptune设备之外，值得注意的是其中还提到两款老式的合成器Omnichord和Quasimidi Sirius
