这篇文章是2018年若干想法的总结，每个话题似乎都可以展开成为一篇完整的文章，但因为时间的缘故无法展开详述了，只留个底给大家头脑风暴用。许多内容截取自我在群里的聊天记录，所以有对话的痕迹。

7月20号：生产C.H.I.P和Pocket CHIP的Next Thing Co.关门了，最后一批没有发出货物被留在香港拍卖。在 @叶师傅 和@子杰 的帮助下，这最后一批货总算进了国内，就看社区能走多远了。

算法艺术实验室：随身携带的口袋黑客机，无时无刻写代码打游戏做音乐

整理一点C.H.I.P和Pocket CHIP的资料：NTC的论坛已经关闭，但博客还可以访问：http://blog.nextthing.co/

Github也可以访问：

Next Thing Co.

爱好者自行开设的Chip社区网站：

http://www.chip-community.org/index.php/Main_Page

Chip本身是开源硬件，包括PCB都是开源的 https://github.com/NextThingCo/PocketCHIP-PCB 但因为C.H.I.P上使用的全志R8应该已经停产了，所以即使想重新生产也很可能需要对原有的设计进行改动。

7月26号：很多时候我觉得，要啥浏览器，有telnet就够了（笑）

Telehack 里面不少好玩的，可以telnet上，对了你们能telnet到 ScramWorks open web clients 吗？

telehack上的字符版2048

ScramWorks open web clients 提供telnet的字符界面浏览器访问，有额外的好处，一般人我不告诉他。

8月5号：有串口的设备就可以拿ESP8266连上wifi，然后telnet到linux主机，然后就能上网了。用ESP8266制作WiFi Modem的教程可以看这里：

http://subethasoftware.com/2018/02/28/wire-up-your-own-rs-232-wifi-modem-for-under-10-using-esp8266-and-zimodem-firmware/

Apple II，C64，MSX这个级别的设备就可以看gopher，刷刷usenet了，毕竟只是当终端运行linux主机上的程序，但Sharp PC-1500的屏幕就太小了，原理上是可以的，不过就不实用了。

RC2014是开源硬件，是开源的Z80电脑，要是国内有寨厂生产就好了：https://rc2014.co.uk/

C64 mini上架了，EB Games和百思买都能买到，虽然C64 mini不是全尺寸的，但可以插USB键盘玩，据说全尺寸版本今年下半年会出。

国内MOS 6581（SID）竟然不是很贵……不知道他们是怎么搞到的……要都是拆机件的话，这要拆多少台C64啊，有兴趣的话可以试一下这个Arduino控制6581的方案：https://github.com/CharlotteGore/MOS6581

国内开展Chiptune的条件很好嘛……要啥有啥，不一定光玩FakeChip嘛，nanoloop在真机上跑是真chip啊，国内gameboy不贵，GB Boy应该是换壳的（评论区里），这个我在温哥华chiptune协会讲座的时候玩到过。

这种mindkids的FC卡带上面是个闪存啊，按道理是可以重新烧录的，有人研究过吗？不知道淘宝上做卡带定制的商家能不能帮忙定制ChipTune卡带。

对FC音乐感兴趣的小伙伴可以看一下这个网站，里面有好几种FC上的音乐工具，你们谁要是会改键位改成用学习机的键盘控制就完美了，然后再烧到卡带里，国产学习机就成了超强的Chiptune工作站。

NES Audio Tools by Neil Baldwin

步步高学习机的BBGDOS是一个真的DOS，不是只有MS-DOS是真的dos的，Apple II有Apple DOS，ProDOS，Amiga有Amiga DOS，Atari ST是GEM DOS；作为一款home computer的dos，步步高的DOS功能是完整的。

8月6号：这个视频里的Elektron SidStation现在已经停产了，二手卖的也很贵，要是能有国产的芯片音乐合成器就好了。

体育场最常听到的Zombie Nation背后的故事

8月12号：蒙特利尔对于demoscene爱好者来说是个不错的地方，可以去Synchrony Demoparty看看：https://synchrony.nyc/2019/index.htm

当然对于美国和加拿大东部的爱好者来说，更值得一看的是Demosplash，规模更大一些

Demosplash 2018 - Welcome

如果在蒙特利尔的话，康考迪亚大学（Concordia University）那边有些时候可能会自己组织一些Computation Arts的展览和活动，你们可以关注下，另外twitter上的 @GlandeurLessard是康考迪亚的老师，也是蒙特利尔demoscene的组织者之一：Jonathan Lessard
