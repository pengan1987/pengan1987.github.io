PowerBook 3400c（1997）初体验

作者：paizhang
知乎文章编号：43459910
创建于：2018-08-31 3:52:45
修改于：2018-08-31 3:52:45

PowerBook全速出击！来自1997年2月的PowerBook 3400c是末代603e PowerPC处理器机型，也是最快的G3前PowerBook机型之一，更是最后一代G3前旗舰笔记本电脑机型。它曾号称世界上最快的笔记本电脑，虽然我所拿到的这台不算——这是一台低配版PowerBook 3400c。

概述

PowerBook 3400c拥有三大亮点：标准的笔记本电脑设计、速度飞快的603e处理器、PCI兼容。标准的笔记本电脑设计令它的扩展性能和可玩性中规中矩，内置的扩展坞可以直接读取CD，无需外置驱动器；速度飞快的603e处理器为Mac OS 7兼容打下基础，也为Mac OS 8带来了不错的性能表现；而PCI兼容则代表它有更好的性能，且可以获得扩展的USB接口，以此进一步提升可玩性。

PowerBook 3400c的「c」意为它具有TFT彩色屏（之前系列还有「cs」型，意为具有DSTN彩色屏）。PB3400系列共拥有三个配置，处理器从180 MHz到240 MHz；这台电脑（PB3400/180）具有180 MHz的603e处理器，性能相对其他版本来说比较差劲，但在与奔腾的竞争中也并未败下阵来；而240 MHz的PB3400则曾经号称世界上最快的笔记本电脑。

  • PowerPC 603eV处理器，180 MHz主频，40 MHz总线速度，16KB+16KB L1缓存，256KB L2缓存；
  • 兼容32位宽PCI架构；
  • 内建16 MB EDO内存，专有内存接口；
  • 无独立显示卡，内建1 MB显示内存，最高可支持1024*768 256色输出；
  • 1.3 GB IDE硬盘，PB3400扩展槽，支持CD驱动器或软盘驱动器；
  • 12.1英寸 SVGA TFT LCD，16位色，分辨率800*600；
  • 支持从Mac OS 7.6（带有PB3400使能器）到Mac OS 9.1，且高度兼容早期Rhapsody系统；
  • 11.5″ * 9.5″ * 2.4″（长宽高），7.2磅（软盘驱动器安装），7.4磅（CD-ROM安装），6.9磅（PC卡安装）。

右侧只有一个扩展槽，我安装了CD-ROM模组。

左侧则依次为ADB、PC卡槽（支持Type I与Type II，仅下槽支持Type III，由于PCI标准，CardBus兼容）、麦克风与扬声器接口。

后侧应该有个祖传挡板，但它丢失了。从左到右依次是红外线、以太网、喜闻乐见的复位钮、SCSI HDI-30、HDI-15苹果显示输出、电源。

设计

PowerBook 3400c带有PowerBook系列一直以来的祖传设计风格，由一个挂钩扣和屏幕与机身，A面的主题则是一个苹果彩色标志与弧面。

PowerBook 3400c的机身尺寸比较小巧，厚度中规中矩。屏幕上左右有两个扬声器，加上键盘左右共四个，带来大声（但音质不佳）的聆听体验。无论如何它的开机声很洪亮。又是一处历史遗留设计，你可以看到右侧的机器支架。它可以通过一个小按钮自如收放，支起整机。

扩展槽可以被弹出，以将它换成别的模组，例如软盘，甚至ZIP磁盘。

扬声器与开机按钮——开机按钮在键盘的右上角。键盘是传统薄膜键盘，按键面积小，反馈十分平淡，让人难以适应。

苹果标志的左右侧宣示这台电脑的型号，下方是麦克风。这代PowerBook将亮度操纵移动至键盘上方，增加声音控制为用户提供方便。

PowerBook 3400c的屏幕边框小，整体观感碾压Mainstreet G3。电源指示灯在屏幕的右上角，电脑睡眠时它会闪动，且在合盖时也可见。合扣下方的「PowerPC」字样暗示这台电脑内部的强劲PPC心脏。

体验

但就如我之前的介绍，我所拿到的这台PB 3400是最低配版，且它在16 MB内存之上运行Mac OS 8.5，自然缓慢至极。因此我已将它降级至Mac OS CH-7.6.1，运行飞快。真机运行Mac OS 7是个有趣的事情，因此本文的体验将全部建立在Mac OS CH-7.6.1的基础之上。

Mac OS 7.6.1是最后的7系统。Mac OS 7是成功的操作系统，也是失败的操作系统。比起System 6，Mac OS 7走入正轨，实现了内建协作式多任务，彩色支持等必要的功能，但功能的堆积使它的稳定性堪忧（而比起之后的版本，却显得原始）。大量M68K模拟代码让PowerPC处理器不能发挥它的完整性能。它的主要竞争对手Windows 95则获得巨大成功，Mac OS开始逐渐淹没在Windows的声誉中。

PowerPC 603eV处理器带来了不错的处理性能。虽然这台电脑的内存严重不足，但内存不够，「虚存来凑」。虚存的性能不好，它会让电脑的运行变缓慢，也吞噬为数不多的硬盘空间；但总比内存不够而无法运行应用程序强。Mac OS 7提供存储器控制板来设置虚存。

PowerBook 3400c系列有严重的电池和PRAM电池泄露问题，但十分幸运的是，虽然这台电脑的PRAM电池失效了——但是它还足够可靠。电脑的主电池不仅还可以正常充电，更能提供高达两个半小时（理论值，在Mac OS下正常使用大约只可用一个半小时）的使用时间。在插上电源时，PowerBook控制板自动将处理器速度调整至性能优先，此时性能显著提高但左掌托处发热严重；断开电源时，则调整为更加节能，此时发热完美改善的同时带来更长的续航时间。

PowerBook 3400c采用了PCI标准，但它本身的设计相对来说比较短命，只沿用了两代，略微悲惨。1997年后，乔布斯归来，且Macintosh们迅速换装具有强大缓存能力的G3处理器；它的直接继承人Kanga G3携250 MHz的PowerPC 750处理器开创了PowerBook的G3时代。Kanga们直接沿用PB3400的模具和标准——Kanga甚至有个昵称，PB3400/G3或PB3400/250，就好像直接替换处理器的3400型——但G3 Kanga也是个短命的机型。G3之后，新款Wallstreet带来了新设计，新模具与SDRAM内存等新概念，且苹果带去了OS X支持，PowerBook系列迅速现代化。PowerBook 3400c们见证了苹果最困难的历史时期，见证了CEO的回归，开辟了新的Macintosh历史。
